# Discord Encrypt Bot
一個利用cryptography函式庫進行檔案加解密的Discord bot

*該機器人並不會記錄或保存任何加解密過的檔案*
# 前置作業
將需要的機器人設定填入config.json中
```
pip install -U -r requirements.txt
```

執行 `main.py`
